# AdventureGame
This project is a text-based adventure game developed over the course of two days as part of a Python programming course.

The application demonstrates the use of core Python concepts, including modular function design, conditional logic (if/else) for game flow control, and while loops for continuous gameplay. It also incorporates input validation using regular expressions via Python’s re module.

Player data is persisted using the json module, allowing the program to store and retrieve scores. Users can view their results alongside previously recorded scores associated with their name.

Additionally, the game utilizes a dictionary structure to track and summarize player decisions throughout each playthrough. At the conclusion of the game, users are prompted to replay, enabling the user to play again if they choose to.
